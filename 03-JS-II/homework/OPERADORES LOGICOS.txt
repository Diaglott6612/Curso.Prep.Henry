OPERADORES LOGICOS

Ahora que sabemos ocupara variables y comparadores de como <; <= etc. podemos usar los operadores lógicos para saber si algo es cierto  no por ejemplo:

	if (15 > 20 && 10 === 10){
		console.log (´Una de estas es cierta por lo que el operador && (´Y´ o ´AND´) no se ejecutara, ya que ambas tendrian que ser verdares
	}
Hay otro operador que es el "O" e "OR" que en Javascript lo escribimos asi "||" hace algo parecido a && con la dieferencia que arrojara true si solo una es verdares ejemplo:

	if (100 < 150 || 20 === 20){
		console.log ( Aqui el programa correra por que hay una que es verdares "20 === 20")
	}

El otro es "NOT" O "NO" que en Javascript se escribe este signo de excalmación "!" el cual negara un aexpresión ejemplo:

	if (!(3 < 2)){
		console.log(´ como 3 no es mayor que 2, este codigo se ejecutara´)

	}

BUCLES for

for es una froma hacer que una operación se repita finitimanete auqnue podria ser infinita, sin embargo el proposito es que cumpla una función y como lo veo todo debe tener un principio y un final. Este Bucle for te ayudara a hacer opreaciones que quizás si las hicieras manualmente serían mjuy tediosas , como contar hasta 100.000 ejemplo:

for (var i = 0 ; i >= 1000000) ; i++){
	console.los(i);
}

Aquí puedes ver que el for permite que una variable llegue hasta "1000000" para que se cumpla 
"i >= 1000000" ¿como lo hizo?. Bueno for se compone de tres cosas:

	1. Variable
	2. Condicional
	3. Incremento de Variable

Como ves en el ejemplo al declarar la variable le asignamos un numero "var i = o" depues asignamos un condicional "i >= 1000000" como "i" no es mayor ni igual a un millon , nos faltaría algo para que nuestra variable "i" pueda ir aumentando y asi cumplir con la exprecion ">=" para eso agregamos "i++" el cual cada vez que de la vuelta irá sumandole uno a nuestra variable hasta que llegue a "i >= 1000000".
