1.VARIABLES
	
	Una variables es como una caja donde podemos guardar nuestros juguetes de manera organizada, para más tarde si las llegasemos a utilizar poder encontrarlas. En Javascript tenemos algunas claves que nos ayudaran a encontar nuestros jueguetes. 
	Hay dos variables principales:
		
		a.La primera es "var" y puede variar, es como decir que en tu caja de juguetes lo que guardarás allí lo puedes cambiar si así lo deseas.
		b.La otra es "const" que es constante, como si los juguetes que tienes en esta caja no quieres que cabien o no pueden ser cambiados.

	Entonces tienes dos cajas una es "var" y la otra es "const" ahora a esto le puedes agregar un nombre más un valor: 
			# var cantidadDeJuguetes = 15;
			# const noSacarJuguetes = 30;

	Asi cuando quieras saber cuantos juguetes tienes en una caja sabras como se llama o como llamarla.

3.FUNCIONES

	Las funciones permitiran trabajar las variables en el caso de la caja de juguetes, podrias modificar los valores, restar, sumar jugutes etc. para eso puedes utilizar una palabra clave de Javascript y es "Function" mas un nombre para la función:

							## function inventarioDeJuguetes (){}

	También se le puede agregar argumnetos como los juguetes rojos y azules que tengas, estos van dentro de los parentesis

			## function calculosConJuguetes (Rojos, Azules){}

	ahora con una variable podras sumar, restar, o dividir tus juguetes azules y rojos, y devolver la operacion con un "return" esto ayuda a que la suma concluya y poder saber el resultado, mirad:

			## function calculosConJuguetes(Rojos, Azules){
					var suma = Rojos + Azules;
					return suma;
			}

	Prueba cambiando en ves de Rojo y Azul, por numeros.
